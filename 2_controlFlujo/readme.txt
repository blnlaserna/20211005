INSTRUCCIONES DE CONTROL DE FLUJO
Permiten alterar la forma de ejecución y que no sea secuencial.
Permite líneas de código que se ejecutan o no en función de una 
condición.
Permiten que líneas de código se ejecuten más de una vez en función
de una condición.
La condición se construye utilizando una expresión lógica.
Una expresión lógica es cualquier tipo de expresión de JS que 
proporcione un resultado booleano.
Las expresiones lógicas se construyen usando variables booleanas,
o bien, escribiendo expresiones que combinen operadores relacionales
y operadores lógicos.
